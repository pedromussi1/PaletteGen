About PaletteGen

Free Color Palette Generator

PaletteGen is a free, open-source color palette generator built for designers, developers, and artists. Create harmonious color schemes based on color theory, generate random palettes, or extract dominant colors from any image — all without creating an account or uploading your data to a server.

Why PaletteGen?

  • 100% Client-Side — Everything runs in your browser. Images for extraction are never uploaded.
  • Color Theory — Six harmony modes: analogous, complementary, triadic, split-complementary, tetradic, monochromatic.
  • Image Extraction — K-means clustering extracts dominant colors from any uploaded image.
  • Lock Colors — Keep specific swatches while regenerating the rest.
  • Developer Export — Copy as CSS variables, Tailwind config, or download as SVG swatches.
  • Shareable Links — Palette encoded in the URL for easy sharing.
  • Free & Open Source — Source code available on GitHub.
